# Digitakt II - Page menu

- In recording mode
- Click Page and upper row trigs to select the trigs apply to, i.e. which bar is show (no-follow)
- Click Page and lower row trigs to select which ones to play at all
- Clear a single track page only
    - In recording mode
    - Select a Track
    - Press [PAGE] to select the page
    - then press [PAGE] + [PLAY] to clear just that page **of the track**
